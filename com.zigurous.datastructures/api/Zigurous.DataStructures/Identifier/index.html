Identifier

Static Class in Zigurous.DataStructures

Declaration

public static class Identifier

Description

Generates identifiers.

Static Methods

GuidCreates a new GUID string similar to the following "0f8fad5b-d9cb-469f-a165-70867728950e".
UnixTimeCreates an identifier based on unix time. Since time is always increasing, this value will be different than generations made in previous cycles. This should not be used to guarentee uniqueness since ids generated within the same cycle will usually be identical.
