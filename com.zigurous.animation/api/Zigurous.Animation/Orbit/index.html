Orbit

Class in Zigurous.Animation

Declaration

public sealed class Orbit : MonoBehaviour

Description

Orbits an object around another object with a given speed and radius.

Properties

centerThe transform the object orbits around.
speedThe speed at which the object orbits.
radiusThe distance from the center of the object being orbited.
startAngleThe initial angle in degrees of the orbiting object.
angleThe current angle in degrees of the orbiting object (Read only).
updateModeThe update mode during which the object orbits.
