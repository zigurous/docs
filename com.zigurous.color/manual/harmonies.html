    Harmonies

    Color harmonies are combinations of colors that create aesthetically pleasing contrasts and consonances that are said to be harmonious. The Color Pro package includes several extension methods declared in the static class Harmonies for generating common color harmonies.

    Monochromatic

    A set of tints and shades formed from a base color.

    // Generate 5 colors with a 25% difference between each tint/shade
Color[] harmony = baseColor.Monochromatic(5, 0.25f);

// Prevent heap allocations by filling an existing array of `output` colors
baseColor.MonochromaticNonAlloc(output, 0.25f);

    Analogous

    A set of colors located next to each other on the color wheel.

    // Generate 3 colors with a 30° hue shift between each color
Color[] harmony = baseColor.Analogous(3, 30);

// Prevent heap allocations by filling an existing array of `output` colors
baseColor.AnalogousNonAlloc(output, 30);

    Complementary

    Two colors located opposite to each other on the color wheel.

    Color[] harmony = baseColor.Complementary();

// Prevent heap allocations by filling an existing array of `output` colors
baseColor.ComplementaryNonAlloc(output);

    Split Complementary

    A base color with two colors adjacent to the directly opposing color on the color wheel.

    Color[] harmony = baseColor.SplitComplementary();

// Prevent heap allocations by filling an existing array of `output` colors
baseColor.SplitComplementaryNonAlloc(output);

    Double Split Complementary

    Two pairs of complementary colors on either side of a base color.

    Color[] harmony = baseColor.DoubleSplitComplementary();

// Prevent heap allocations by filling an existing array of `output` colors
baseColor.DoubleSplitComplementaryNonAlloc(output);

    Triadic

    Three colors evenly spaced around the color wheel to form a triangle (120° hue shifts).

    Color[] harmony = baseColor.Triadic();

// Prevent heap allocations by filling an existing array of `output` colors
baseColor.TriadicNonAlloc(output);

    Square

    Four colors evenly spaced around the color wheel to form a square (90° hue shifts).

    Color[] harmony = baseColor.Square();

// Prevent heap allocations by filling an existing array of `output` colors
baseColor.SquareNonAlloc(output);
