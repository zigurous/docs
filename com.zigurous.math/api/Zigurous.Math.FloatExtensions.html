    SAFE_FLOAT

    Sometimes Unity throws precision errors for really small numbers such as when setting transform values. Using float.Epsilon can still cause issues so this value is intended to be a safer alternative.

    Declaration
    public const float SAFE_FLOAT = 1.4E-32F
